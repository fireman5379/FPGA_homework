# FPGA_homework
Basys3 Snake Game
需要添加BLOCK MEMORY IP核 coe文件放在了'coe'文件夹中，
protected,grade,slowly为奖励信息图片
320.coe 为320*240的封面图片
Snake.bmp为封面图片
通过BMP2Mif将图片转化为coe文件
